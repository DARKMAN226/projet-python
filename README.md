## Présentation

Cette application Quiz desktop est développée en Python avec la bibliothèque Tkinter pour l’interface graphique. Elle offre une expérience de quiz multi-catégories et multi-niveaux, avec des questions chronométrées, gestion des utilisateurs et suivi des scores.
